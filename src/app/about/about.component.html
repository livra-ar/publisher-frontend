About

Livra, the mobile application and this accompanying web application was created for the Software Engineering Project module in the Semester 5 of Department of Computer Science and Engineering, University of Moratuwa. This web application allows interested book publishers to add augmented reality content related to their books, which can then be downloaded and viewed by users using the Livra mobile application.

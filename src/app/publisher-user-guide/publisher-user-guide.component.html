Web Platform

You can add books and content to the web platform which can be used by all the users of the system. At first you have to create an account as a publisher.

Registration

  1. Go to our Livra web platform by clicking the following link. https://ar-content-platform.github.io/
  2. Click the ‘Publishers’ button on the top right hand corner and select ‘register’.
  3. Add your details according to the given descriptions and click Register.
  4. Login to the account with those credentials.

Adding a book

  1. Click the ‘Add book’ option on the top navigation panel.
  2. Fill the required details about the book.
  3. You can add up to 3 images, but the first image will be used as the main cover image of the book. Then add the book to the platform.

Adding content

  1. Click the ‘Add content’ option right next to the previous one and fill the details.
  2. You can link your content only with a book added through your account.
  3. The content must have at least 100 character long description.
  4. If the content you are uploading has embedded animation, check the ‘Animated?’ checkbox at the bottom.
  5. Same as books the first image of the three images will be used as the cover image for that content.
  6. Currently all the content you upload should be in the format of ‘.sfb’ for our app to recognize and render your content successfully.
  7. Now you can add the content.
